# Pomodoro Timer  `(ptimer)`

A command-line interface (CLI) pomodoro timer application built in Go that helps you boost productivity with customizable focus sessions.

---

## Features

### 1. Three Pomodoro Modes

Each mode comes with pre-configured **Work**, **Short Break**, and **Long Break** durations tailored to different focus needs:

| Focus Mode | Work Duration | Short Break | Long Break |
| ---------- | ------------- | ----------- | ---------- |
| **Quick**  | 25 min        | 5 min       | 20 min     |
| **Deep**   | 50 min        | 10 min      | 25 min     |
| **Turbo**  | 90 min        | 15 min      | 30 min     |

**Work Session** - Focus time with progress tracking.

**Short Break** - Quick rest between work sessions

![pomodoro2_short_break](https://raw.githubusercontent.com/mithun1st/ptimer/master/screenshots/pomodoro2_short_break.png)

 **Long Break** - Extended break after completing multiple sessions
 
![pomodoro3_long_break](https://raw.githubusercontent.com/mithun1st/ptimer/master/screenshots/pomodoro3_long_break.png)


### 2. Notification Support

Stay informed with cross-platform notifications when your timer completes:
- Silent Mode - Focus without interruptions
- Desktop Notifications - Native system alerts
- Sound Alerts - Audio cues for session changes

![pomodoro4_notification](https://raw.githubusercontent.com/mithun1st/ptimer/master/screenshots/pomodoro4_notification.png)


### 3. View Log

Track your productivity with detailed session logs:
- Completed cycles
- View total work time
- Short break time
- Long break time
- Statistics Visual Progress

![view_log](https://raw.githubusercontent.com/mithun1st/ptimer/master/screenshots/view_log.png)


### 3. Quick Timer With Flag

Start timers instantly from the command line with custom durations:
- `-m` for minutes
- `-h` for hours
- `-s` for seconds
- `--notify` Option to enable/disable notifications 
- `--color` Option to enable/disable notifications
